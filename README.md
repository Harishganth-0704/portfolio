### 🌟 Advanced Features Built-in:
1. **Premium Glassmorphic Design**: Leverages heavy backdrop filters (`backdrop-filter: blur(20px)`), glowing semi-transparent borders, and radial background gradients to deliver a stunning modern dark cyber theme.
2. **Dynamic Project Filtering**: Employs client-side JavaScript DOM manipulation to instantly filter projects between **AI-Driven** and **Full-Stack** tags with zero loading delays.
3. **Simulated GitHub Heatmap Board**: Dynamically generates a realistic 371-cell green contribution grid (53 columns x 7 rows) using JavaScript on page load to mock live development activity.
4. **Intersection Scroll Observer**: Automatically highlights the active section in your sticky navigation header as you scroll down the page, providing a premium native-app feel.
5. **Interactive Chronological Timeline**: Showcases hackathon team achievements (Elite Hack 1.0 Finalist), national essay competitions, and professional credentials on a linear glowing track.
6. **Wired Action CTAs**: Instant-response contact cards and a mock message form alongside a functional HTML5 **Resume Download** button mapped to a local file trigger.

## 🛠️ Local Installation & Development

To clone, run, and check this project locally on your machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Harishganth-0704/portfolio-main.git
   cd portfolio-main/portfolio-main
   ```

2. **Start a local development server** with caching disabled:
   * **Using Node (Recommended)**:
     ```bash
     npx http-server -p 8081 -c-1
     ```
   * **Using Python**:
     ```bash
     python -m http.server 8081
     ```

3. **View the website**:
   Open your browser and navigate to `http://127.0.0.1:8081`
